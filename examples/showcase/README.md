# czap showcase

> **New to LiteShip?** This is the everything-at-once demo, not the starting point.
> Climb the [examples ladder](../README.md) first — it builds up to what's shown here.

Feature showcase for the worker, GPU, stream, and LLM client directives.

When installing from npm (outside the monorepo), pin `@czap/*` packages at `^0.8.0`.

```bash
pnpm install
pnpm dev
```
