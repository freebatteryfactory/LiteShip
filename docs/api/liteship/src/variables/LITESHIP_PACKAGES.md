Defined in: [liteship/src/index.ts:36](https://github.com/freebatteryfactory/LiteShip/blob/main/packages/liteship/src/index.ts#L36)

Every `@liteship/*` package this umbrella installs, in dependency order.
Consumed by audit/doctor/release tooling that needs the canonical fleet
list; app authors never need to import it for layers 1–3.

This is the tarball-shipped mirror of the canonical `@liteship/*` fleet roster
owned by `scripts/gen-roster.ts` (`CANONICAL_ROSTER`). The const stays local
because `liteship` ships in the published tarball and cannot import a
dev-only script; the `BEGIN/END gen-roster` block below is regenerated by
`pnpm tsx scripts/gen-roster.ts` (its `renderLiteshipPackages()` output) and
the `liteship-packages-roster` drift-guard byte-compares it against that
generator, so the two can never silently diverge.

The single fleet-roster OWNER these names re-anchor to is `@liteship/audit`'s
`LITESHIP_PACKAGE_ROSTER` (the scoped `@liteship/*` fleet). This copy stays local by the
SAME shipping constraint — `liteship` ships in the tarball and cannot import the
devops-layer `@liteship/audit` — and its parity with that anchor is held by the
`liteship-packages-roster` drift-guard, not a shared import.
