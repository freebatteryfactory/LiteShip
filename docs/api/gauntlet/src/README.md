# gauntlet/src

`@liteship/gauntlet` — the self-proving, extendable rigor engine.

The foundations (Slice A): the [Finding](interfaces/Finding.md) vocabulary humans and agents
share, [AssuranceLevel](type-aliases/AssuranceLevel.md) (the hazard model that aims rigor), the
[Gate](interfaces/Gate.md) plugin contract (a fitness function `(context) => Finding[]`),
and the authority ratchet (a gate earns blocking power by self-proving
against its own red/green/mutation fixtures — see [verifyGate](functions/verifyGate.md)).

Extend by composing: a downstream project `defineGate`s its own gate with its
own fixtures and hands it to [runGates](functions/runGates.md) alongside LiteShip's built-ins —
no fork, no rebuild. The engine qualifies every gate, theirs and ours, by the
same ratchet.

Slice B widens [GateContext](interfaces/GateContext.md) into the triangulated repo-IR; Slice C adds
the mutation/simulation/avionics gate families. The contracts here do not
change underneath them.
