# Timeline

Timeline — scheduler-driven advancement over a [Boundary](../../variables/Boundary.md).
Produces a plain reactive timeline that seeks or plays between boundary states;
pluggable clock via [Scheduler](../../variables/Scheduler.md), teardown via [Lifetime](../../variables/Lifetime.md).
